# UE4 - ELISA Daten

Contains the analysis notebook and the accident data used for this exercise.

## Data files

- `UE4_ELISA.ipynb` - analysis notebook
- `Unfalldaten A5 2024.xlsx` - accident data
- `T1_2023_KW35_2023-09-01_bis_2023-09-01.csv`, `T2_2023_KW35_2023-09-01_bis_2023-09-01.csv` - sample weekly traffic count data (KW35)

The remaining weekly traffic count CSVs (KW36-KW39, T1 and T2) are not included here because they exceed GitHub's 100 MB file size limit (up to ~210 MB each, ~1.3 GB total). Ask for the original files if you need the full dataset.
